# Vision and MVP

## Vision
Open the app and instantly see:
- Which nearby NL spots are **sendable**
- The **best upcoming window** (start–end + duration)
- A transparent **SendScore (0–100)** and “why”

## Target user
Intermediate kitesurfers.

## MVP promise
**Nearby wind → SendScore → best window → optional alert.**

## MVP features
- NL spot list (curated JSON)
- Geolocation (“around me”) + radius filter
- Hourly forecast per spot (24–48h)
- SendScore 0–100 + Green/Yellow/Red
- Best upcoming window per spot (Option A: show only best window)
- Alerts (rule-based; delivery can start simple)

## Non-goals (MVP)
- Session logging
- Social features
- Waves/tides (can come later)
- Complex safety modeling (keep simple + explicit overrides)
