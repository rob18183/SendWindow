# Roadmap

## Milestone 0: UI-first prototype
- Mock forecasts
- Home list + sorting
- Spot detail timeline

## Milestone 1: Real forecast provider
- Replace mock with real hourly wind + gust + direction
- Add caching (localStorage, 15 min TTL)

## Milestone 2: Alerts
- Rule evaluation
- Basic delivery (email/telegram/push later)

## Milestone 3: Spot quality
- Improve good/unsafe wind ranges per spot
- Add spot notes (parking, hazards)

## Milestone 4: Personalization
- Rider profiles (beginner/intermediate/advanced)
- Adjustable thresholds

## Later
- Waves/tides
- Community edits
- Session logging
